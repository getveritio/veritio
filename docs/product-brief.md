# Veritio Product Brief

## Positioning

Veritio is the open-source evidence layer for software teams that need application-level audit trails, consent history, DSAR workflows, records of processing, retention policies, and compliance exports without adopting a full GRC platform.

## Primary Users

- Developers who need a small SDK and clear event contract.
- Product teams that need customer-visible audit and privacy workflows.
- Security and compliance teams that need verifiable, exportable evidence.
- AI coding agents that need stable schemas and integration guides.

## OSS Promise

- Install locally.
- Self-host the server.
- Keep events in your own database.
- Use language SDKs without a hosted account.
- Export evidence without proprietary lock-in.

## Provider Promise

The hosted Veritio offering can sell:

- managed event ingestion and storage
- retention automation
- SIEM exports
- customer audit portals
- DSAR request workflows
- evidence bundles for sales and security reviews
- EU-region hosting and operational guarantees

## First Product Boundary

The first version focuses on the event protocol and tamper-evident audit records. Consent, DSAR, records of processing, SIEM export, and UI modules build on the same protocol.
